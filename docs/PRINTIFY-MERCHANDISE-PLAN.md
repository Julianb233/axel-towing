## 1. Strategic Overview

The merchandise store serves two parallel purposes:

1. **Corporate Gifting (Primary):** Branded gifts for HOA offices and property managers to build relationships and get foot-in-the-door — the "gift before the pitch" strategy.
2. **Customer-Facing Merch (Secondary):** Public-facing store on axletowing.com for brand visibility and minor revenue.

The print-on-demand model (via Printify) means zero upfront inventory cost. Products are manufactured and shipped only when ordered. For corporate gifting in bulk, Printify also supports bulk ordering at reduced per-unit cost.

## 3. Printify Setup Guide

### Step 1: Create Printify Account

1. Go to https://printify.com/app/register
2. Create account with Elliott's email
3. Select "I want to sell online" — choose "My own website" as the platform
4. Complete merchant profile with business name: "Axle Towing & Impound"

### Step 2: Connect to Website

Printify supports multiple integration options:

**Option A: Printify API (Recommended for axletowing.com)**

- Use Printify API to embed a lightweight store on axletowing.com
- API docs: https://developers.printify.com/
- Requires a simple Next.js `/merch` page that calls the Printify catalog API
- Customers checkout via Printify's hosted checkout (no payment processing required on our end)

**Option B: Shopify Integration**

- Create a Shopify Lite store ($9/month) → connect Printify
- Embed Shopify Buy Button on axletowing.com
- Easier setup, but adds monthly cost

**Option C: Manual Ordering Only (Easiest for Corporate Gifts)**

- No public-facing store needed initially
- Elliott places orders directly through Printify dashboard
- Enter recipient HOA office address manually
- Products ship directly to HOA office with custom gift message

**Recommendation:** Start with Option C (manual) for the corporate gifting program. This takes 30 minutes to set up and allows immediate gifting. Add the website storefront (Option A or B) as a Phase 2 upgrade.

### Step 3: Upload Designs

1. Navigate to Product Catalog in Printify
2. For each product: select product → customize → upload artwork
3. Design specs for each product are listed in Section 2 above
4. Create a "Product Collection" called "Corporate Gifting — HOA" with gift items
5. Create a second collection "Customer Merch" for t-shirts, hats, stickers

**Design assets needed:**

- High-res Axle Towing logo (SVG or PNG at 300+ DPI) — get from Elliott
- Brand colors: Dark Navy (#1B2A3F), Red (#DC2626), Blue (#1E6BB8)
- Secondary tagline options:
  - "Phoenix's Private Property Towing Specialists"
  - "24/7 Parking Enforcement | (480) 288-5526"
  - "Serving Phoenix Metro Since [year]"

### Step 4: Configure Shipping

For corporate gifting:

- Printify ships directly to the recipient address
- Default shipping: USPS First Class (~3–5 business days)
- Expedited: USPS Priority (~2 business days, +$3–5)
- No return address issues — customize the "from" name in Printify settings to "Axle Towing & Impound"

### Step 5: Test Order

Before sending gifts to real HOA contacts:

1. Place a test order for 1 mug + 1 mouse pad
2. Ship to Elliott's address
3. Review quality — check logo positioning, print quality, color accuracy
4. Adjust designs if needed before bulk ordering

## 6. Ordering Workflow

### Workflow A: Individual Corporate Gift (High-Touch, Score 7–10 Leads)

1. Identify lead from the HOA prospect list (AI-2026 database)
2. Look up office mailing address (from Apollo.io or Google Maps)
3. Select gift tier based on lead score:
   - Score 7–8: Mug or Mouse Pad ($12–25)
   - Score 9–10: Wireless Charger or Full Kit ($26–50)
4. Log in to Printify dashboard → Create order
5. Ship to HOA office address
6. Add gift note: "Thinking of [Community Name] — we'd love to discuss how Axle Towing can support your parking enforcement needs."
7. 3–4 days after estimated delivery → follow up by email/phone
8. Log gift in GHL contact record: tag `gift-sent-[date]`

### Workflow B: Bulk Gift Campaign (Monthly Batches, 20–50 HOAs)

1. Pull list from GHL: all `priority-hot` or `priority-warm` leads not yet contacted
2. Export addresses to spreadsheet
3. Use Printify's bulk order feature (CSV upload for multiple recipient addresses)
4. All orders ship simultaneously — same delivery window
5. Schedule follow-up sequence in GHL: trigger email Day 5, call Day 7

### Workflow C: Customer Order (Public Store)

1. Customer visits axletowing.com/merch
2. Selects product, pays through Printify checkout
3. Printify handles fulfillment and shipping automatically
4. Elliott receives order notification via email
5. Zero manual work required
