# Axle Towing — HARO / Connectively / Featured.com Press Pitches

5 ready-to-deploy pitch templates for journalist queries on towing, parking, abandoned vehicles, HOA enforcement, and Phoenix property management topics. All bylined to **Elliott (Owner)** as primary expert source. Brian (Operations, 25+ yrs) is secondary attribution for operational details.

## How to use

1. Monitor 3 platforms daily (queries posted 3x/day):
   - **Connectively** (formerly HARO): https://www.connectively.us/ (free tier OK)
   - **Featured.com**: https://featured.com/experts (paid $39/mo — high quality matches)
   - **Qwoted**: https://www.qwoted.com/ (free tier)
2. Filter for keywords: `towing`, `tow truck`, `abandoned vehicle`, `parking enforcement`, `HOA parking`, `apartment parking`, `private property towing`, `Arizona towing`, `Phoenix property management`
3. When a relevant query lands, customize the closest template below with the SPECIFIC angle the journalist asked for. Don't send generic — journalists detect it instantly.
4. Send within 4 hours of posting (most queries close in 24h, best responses arrive in first few hours).
5. Log every pitch sent in `pitches-sent.log` (created on first send).
6. When picked up, log the publication URL in `placements.md` and add a backlink check to monthly Linear.
